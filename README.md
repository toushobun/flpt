# FLPT

## 概要
大学の卒業制作として作成した Java Web アプリケーションです。
Spring MVC をベースに、基本的なWebアプリケーションの構成（画面・ロジック・データ連携）を学習する目的で開発しました。

ユーザー操作に応じたデータ処理や画面表示を行う、シンプルな業務系アプリケーションの構成を意識しています。

## 使用技術
- Java
- Spring MVC
- JSP / Servlet
- HTML / CSS / JavaScript
- MySQL（または使用しているDBに合わせて書き換えてOK）
- Tomcat

## 主な機能
- ユーザー入力に応じたデータ処理
- 画面遷移制御（Controller経由）
- データの登録 / 表示 / 更新 / 削除（基本的なCRUD）
- フォーム入力とサーバーサイド処理の連携
- MVC構成による責務分離

## システム構成
- Controller：リクエスト処理・画面遷移制御
- Service / Logic：業務ロジック
- DAO：データアクセス処理
- View（JSP）：画面表示

## 実行方法
1. Java / Tomcat を準備
2. データベースをセットアップ（必要に応じて）
3. プロジェクトを Tomcat にデプロイ
4. ブラウザからアクセス

※環境構築方法は開発環境に依存します

## 補足
- 大学の卒業制作として作成した個人開発プロジェクトです
- 学習目的のため、構成はシンプルです
- 基本的なWebアプリケーション開発の流れ（MVC構成）を理解することを目的としています
